April 4, 2021 | ARKADATA | Audio Production

I’m fascinated by simple pieces that only use a string ensemble. No synths, no brass, no winds, no percussions.
It’s a great exercise to reproduce such arrangements, especially when considering how much work can be put into production.

Trying to reproduce a Japanese Orchestra with Cinematic Studio Strings.

Here is the original soundtrack from Sawano Hiroyuki. It’s a 3min54 strings piece, with a classical feel.

ARKADATA · kanascene

The first step to reproduce the piece was to get the main instruments and timing right. Starting with the violin as our leading voice.

ARKADATA · # Kanascene Azkver2

I tried to get as close as possible with Cinematic Studio Strings, but the ensemble is a bit bigger than what was used on Aldnoah Zero’s soundtrack. Which used a Japanese studio orchestra (Muroya Strings), which is probably half the size of what I’m currently using.

A way I found to make up for that is to use solo strings in addition to the ensemble. It gives more definition to the sound.

ARKADATA · # Kanascene Azkver10

It’s getting there but it still feels a bit stiff.

Some more work on the Cello to get all the notes and timing right. 

I used a convolution reverb alongside an algorithmic one to get the sound I wanted. Which consists of an MCO5 IR (Studio Room) with differents mics placements and some FabFilter Pro-R.

Here’s the final result:

ARKADATA · # Kanascene <azk.ver>

The colors are quite different from the original, but I feel like the production value is quite there. It sounds.
Slightly more powerful, less intimate, probably due to the libraries being quite bigger in size than a traditional Japanese orchestra.
